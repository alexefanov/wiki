---
description: 'Запуск эксперимента (slug) [--review] [--collect] [--full] [--env local|remote]'
---

Запусти полный эксперимент, используя скилл `exp-run`.

Аргументы: $ARGUMENTS

Если аргументы пусты — спроси пользователя:
- «Какой эксперимент запускать? Укажи slug»

## Флаги

- `--review` — показать план перед запуском для подтверждения
- `--collect` — собрать результаты из удалённой среды (tools/remote.py collect)
- `--full` — полный цикл: запуск + сбор + оценка
- `--env` — среда выполнения:
  - `local` (по умолчанию) — на локальной машине
  - `remote` — на удалённом сервере (tools/remote.py)

## Примеры

```
/exp-run exp-2024-001
/exp-run exp-2024-001 --review --env remote
/exp-run exp-2024-001 --full
/exp-run exp-2024-001 --collect
```

Порядок действий:
1. Если slug пустой — спроси у пользователя
2. Определи флаги
3. Вызови скилл `exp-run` с slug и флагами
