Проверь целостность базы знаний, используя скилл `check`.

Аргументы: $ARGUMENTS

## Флаги

- `--json` — вывод в формате JSON (tools/lint.py --json) для парсинга другими инструментами
- `--fix` — автоматическое исправление найденных проблем (tools/lint.py --fix)
- `--suggest` — вывод рекомендаций по улучшению (без исправлений)

## Примеры

```
/check
/check --json
/check --fix
/check --suggest
/check --fix --suggest
```

Порядок действий:
1. Определи флаги из аргументов
2. Если флагов нет — запусти обычную проверку
3. Вызови скилл `check` с определёнными флагами
