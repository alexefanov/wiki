---
description: 'Вопрос к wiki [--crystallize] [--format table|timeline|bullets]'
---

Спроси у системы вопрос, используя скилл `ask`.

Аргументы: $ARGUMENTS

Если аргументы пусты — спроси пользователя:
- «Какой у тебя вопрос к базе знаний?»

## Флаги

- `--crystallize` — записать ответ как новую страницу в wiki (вызовет write_back в tools/research_wiki.py)
- `--format` — формат вывода ответа:
  - `markdown` (по умолчанию) — обычный markdown
  - `table` — таблица markdown
  - `timeline` — временная шкала
  - `bullets` — список буллетов

## Примеры

```
/ask Что такое обследование зданий?
/ask --crystallize --format table Какие методы неразрушающего контроля существуют?
/ask --format bullets Кратко об основных нормативах по экспертизе
```

Порядок действий:
1. Определи, есть ли флаги в аргументах
2. Извлеки вопрос (текст без флагов)
3. Если вопрос пустой — спроси у пользователя
4. Вызови скилл `ask` с извлечённым вопросом и флагами
