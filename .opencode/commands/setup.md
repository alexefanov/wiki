---
description: 'Настройка API-ключей'
---

Настрой API-ключи для внешних сервисов, используя скилл `setup`.

Аргументы: $ARGUMENTS

## Описание

Команда запускает интерактивную настройку API-ключей:
- Semantic Scholar API (для поиска статей)
- OpenReview API (для рецензий)
- ArXiv API (для ленты статей)
- Email SMTP (для дайджестов)

Настройка сохраняется в файле `.env` в корне проекта.

## Примеры

```
/setup
```

Порядок действий:
1. Вызови скилл `setup`
2. Следуй интерактивным подсказкам для ввода API-ключей
