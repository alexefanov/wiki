Найди релевантные статьи для исследования, используя скилл `discover`.

Аргументы: $ARGUMENTS

Если аргументы пусты — спроси пользователя:
- «Что искать? Укажи тему (--topic), якорную статью (--anchor) или используй --from-wiki»

## Режимы поиска (выбери один)

- `--anchor <id>` — найти статьи, похожие на указанную (можно указать несколько: --anchor A --anchor B)
- `--topic "<строка>"` — поиск по теме на естественном языке
- `--from-wiki` — автоматически взять якори из текущего состояния wiki
- `--venue <slug> --year <int>` — поиск по конкретному конференц/журналу и году

## Флаги

- `--limit N` — максимальное количество результатов (по умолчанию 10)
- `--negative <id>` — исключить статьи, похожие на указанные (можно несколько)

## Примеры

```
/discover --topic "трансферное обучение в компьютерном зрении"
/discover --anchor paper-2024-transformer --limit 5
/discover --from-wiki
/discover --venue neurips --year 2024 --limit 20
/discover --anchor A --anchor B --negative C
```

Порядок действий:
1. Определи режим поиска из аргументов
2. Если ни один режим не указан — спроси у пользователя
3. Вызови скилл `discover` с указанными параметрами
