---
description: 'Related Work (тема) [--format latex|markdown] [--max-papers N]'
---

Сгенерируй раздел Related Work, используя скилл `survey`.

Аргументы: $ARGUMENTS

Если аргументы пусты — спроси пользователя:
- «Какая тема для обзора литературы?»

## Флаги

- `--format` — формат вывода:
  - `latex` (по умолчанию) — LaTeX-раздел для статьи
  - `markdown` — markdown-формат для wiki
- `--max-papers N` — максимальное количество статей в обзоре (по умолчанию 30)

## Примеры

```
/survey трансферное обучение в компьютерном зрении
/survey "методы сегментации трещин" --format markdown --max-papers 20
/survey --format latex --max-papers 50
```

Порядок действий:
1. Если тема пустая — спроси у пользователя
2. Определи флаги
3. Вызови скилл `survey` с темой и флагами
