2. Оценивать источник по области применения (копия в вики)
3. Добавить свойство Заменен (дата) и Чем заменен (название документа = название файла в raw)
4. Процесс подготовки источника

	2. все блоки текста без заголовков - пометка идентификатором
	3. добавить содержание
	4. добавить историю версий
