# Экспресс-обследование по внешним признакам

Упрощённая оценка технического состояния и надёжности строительных конструкций **по внешним признакам** без полного инструментального обследования. Основана на «Рекомендациях по оценке надёжности строительных конструкций зданий и сооружений по внешним признакам» (ЦНИИПромзданий).

**Уверенность:** средняя (обзор по началу raw; не заменяет ГОСТ при ответственных решениях).

**Границы применимости:** быстрый предварительный отбор объектов, нуждающихся в детальном обследовании. **Не** отменяет обязательного обследования по ГОСТ 31937 там, где оно требуется нормами.

**Ограничения:** метод не для уникальных сооружений, зданий с динамическими нагрузками, после ЧС без последующей верификации инструментальными методами. Не позволяет определить остаточный ресурс в численных значениях.

## Входы процесса

- Нормативные документы: таблицы дефектов и критерии надёжности для каждого типа конструкций (сталь, ЖБ, камень, дерево)
- Предшествующее обследование: актуальные данные о здании (год постройки, проект, этажность, конструктивная схема)
- Доступ к конструкциям: возможность визуального осмотра несущих элементов (колонн, балок, ферм, стен, перекрытий)

## Ресурсы

- **Персонал:** инженер с опытом визуального обследования (знание характерных дефектов для каждого типа конструкций)
- **Оборудование:** бинокль, рулетка, щуп для трещин, уровнемер, фотоаппарат, журнал осмотра
- **Документация:** таблицы дефектов по типам конструкций (из Рекомендаций ЦНИИПромзданий)

## Пошаговая инструкция

1. **Изучение проектной и эксплуатационной документации.**
   - Конструктивная схема, расчётные нагрузки, материалы.
   - История эксплуатации: ремонты, усиления, перепланировки, аварийные ситуации.
2. **Визуальный осмотр несущих конструкций.**
   - Последовательный обход здания: фундаменты (цоколь), стены, колонны, ригели, перекрытия, покрытия, связи.
   - Фиксация всех видимых дефектов: трещины (раскрытие, протяжённость, направление), прогибы, отклонения от вертикали, коррозия, выколы бетона, гниль древесины.
   - Зарисовка схем расположения дефектов на поэтажных планах.
3. **Сравнение с таблицами дефектов по типу конструкций.**
   - Для каждого типа (сталь, ЖБ, камень, дерево) — определение категории дефекта по степени опасности.
   - Если дефект соответствует одной из строк таблицы — фиксируется категория надёжности.
4. **Оценка надёжности каждой конструкции.**
   - На основе таблиц: нормальная → удовлетворительная → неудовлетворительная → аварийная.
   - Для здания в целом — по наихудшей конструкции данного типа (несущий остов).
5. **Формулирование заключения.**
   - Если надёжность неудовлетворительная или аварийная → рекомендовать детальное инструментальное обследование по ГОСТ 31937.
   - Если нормальная или удовлетворительная → установить срок следующего планового осмотра.

## Выходы процесса

- Акт (карта) визуального осмотра с указанием дефектов и их расположения
- Заключение о предварительной категории надёжности каждой конструкции и здания в целом
- Рекомендация: допуск к эксплуатации, плановый осмотр, либо необходимость детального обследования

## Оценка успешности

- Полнота охвата: осмотрены все доступные несущие конструкции (не менее 80% по площади/количеству)
- Дефекты зафиксированы с привязкой к планам и фотографиями
- Категория надёжности присвоена каждой несущей конструкции

## Если результаты не достигнуты

- **Недостаточный доступ к конструкциям** → частичное вскрытие, отбой молотком, шурфы (см. [[Комплексное обследование зданий и сооружений]])
- **Неоднозначная интерпретация дефекта** → инструментальное подтверждение (измерение прочности, УЗК, вскрытие арматуры)
- **Выявлены дефекты категории «аварийная»** → немедленный вывод людей, ограждение опасной зоны, внеочередное детальное обследование
