# Определение причин дефектов каменных конструкций

Подпроцесс судебной строительно-технической экспертизы по установлению причин возникновения и развития дефектов каменных несущих и ограждающих конструкций (прежде всего наружных стен).

**Авторы:** В.Б. Будько, И.Ю. Грунин, Д.С. Дубровский, А.В. Макеев, Т.Е. Троицкий-Марков, С.А. Щигрев.

**Уверенность:** средняя.

**Границы применимости:** каменные и армокаменные несущие и ограждающие конструкции зданий и сооружений.

## Входы процесса

- Дефектная ведомость (акт осмотра) с фотофиксацией
- Проектная документация: конструктивные решения (раздел КР/КЖ), расчётные нагрузки, грунты основания
- Исполнительная документация: акты скрытых работ, журналы производства работ, паспорта материалов (кирпич, раствор, арматура)
- Данные геомониторинга (при наличии деформаций основания)
- Материалы дела: претензии, иски, хронология событий

## Ресурсы

- **Персонал:** эксперт-строитель (конструктор), геотехник (при деформациях основания)
- **Оборудование:** рулетка, штангенциркуль, щуп для трещин, маяки на трещины (гипсовые/электронные), нивелир, теодолит, склерометр (для раствора), УЗ-прибор (для прочности камня)
- **Лаборатория:** прочность кирпича/камня, прочность раствора, химический анализ раствора (сульфаты, хлориды)

## Пошаговая инструкция

1. **Изучение материалов дела и документации.**
   - Анализ проекта: конструктивная схема, марки кирпича/камня, марки раствора, армирование.
   - Хронология: когда построено, какие события предшествовали дефектам.
2. **Анализ факторов, способных вызвать дефекты (по 7 факторам).**
   - Осадочные процессы в грунтах основания (проверить геологию, соседние стройки).
   - Силовое воздействие, не предусмотренное проектом (надстройка, перепланировка, изменение нагрузок).
   - Ошибки проектирования (недостаточное армирование, отсутствие деформационных швов).
   - Нарушения строительных норм при производстве работ (некачественная перевязка, толстые швы, несоответствие раствора).
   - Низкое качество материалов (кирпич ниже проектной марки, расслоение, морозное разрушение).
   - Внешние воздействия (пожар, взрыв, вибрация, сейсмика).
   - Эксплуатационные факторы (замачивание, промерзание, химическая агрессия).
3. **Причинно-следственный анализ.**
   - Для каждого дефекта установить тип возможной причины (первичная / вторичная / сопутствующая).
   - Использовать логические цепочки: просадка основания → неравномерная осадка → трещина в стене → раскрытие через растворные швы.
   - Если дефектов много → построить матрицу «дефект → возможные причины → доказательства».
4. **Инструментальная верификация.**
   - Измерение раскрытия трещин, их протяжённости, направления.
   - Контроль прочности камня и раствора (керны/склерометрия).
   - Геодезический контроль вертикальности и осадок.
   - Установка маяков для мониторинга динамики.
5. **Лабораторные исследования (при необходимости).**
   - Химический анализ раствора (сульфатная коррозия, выщелачивание).
   - Петрографический анализ кирпича/камня (морозное разрушение, примеси).
6. **Формулирование выводов.**
   - Причина дефекта: усадка основания, ошибка проекта, низкое качество работ/материалов, внешнее воздействие.
   - Если причин несколько — степень влияния каждой (основная / сопутствующая).
   - Рекомендации: усиление (см. [[СП 427.1325800.2018]]), мониторинг, ремонт.

## Выходы процесса

- Матрица «дефект → возможные причины → подтверждающие признаки»
- Протоколы инструментальных и лабораторных исследований
- Заключение эксперта с установлением причинно-следственных связей
- Рекомендации по устранению и предотвращению развития дефектов

## Оценка успешности

- Для каждого дефекта установлена причина (или обоснованно указана неопределённость)
- Причинно-следственная связь подтверждена данными инструментального контроля или лаборатории
- Выводы ранжированы по степени влияния (основная/сопутствующая причина)

## Если результаты не достигнуты

- **Скрытые дефекты в теле стены** → зондаж, эндоскопия, частичное вскрытие.
- **Невозможность отбора образцов (историческое здание, памятник)** → неразрушающие методы с понижением уверенности.
- **Противоречивые данные (несколько равновероятных причин)** → вывод о нескольких возможных причинах с ранжированием по вероятности.
