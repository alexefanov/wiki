# Об осуществлении замены элементов конструкций при текущем ремонте

Подпроцесс судебной строительно-технической экспертизы по разграничению понятий текущего и капитального ремонта применительно к замене и восстановлению отдельных элементов строительных конструкций, систем инженерно-технического обеспечения и сетей.

**Уверенность:** средняя.

**Границы применимости:** споры о квалификации ремонта (текущий vs капитальный) при экспертизе сметной документации, актов КС-2, договоров подряда.

## Входы процесса

- Договор подряда (техническое задание, виды работ)
- Сметная документация (ведомость объёмов, локальные сметы)
- Акты КС-2, КС-3, КС-11 с перечнем выполненных работ
- Дефектная ведомость (при плановом ремонте)
- Проектная документация (для определения несущей способности конструктивной схемы)

## Ресурсы

- **Персонал:** эксперт-строитель (с опытом эксплуатации и ремонта зданий)
- **Оборудование:** не требуется (кабинетная работа по документации), при необходимости — фотоаппарат для фиксации

## Пошаговая инструкция

1. **Изучение перечня работ, подлежащих квалификации.**
   - Выделить работы по замене/восстановлению элементов конструкций, инженерных систем.
   - Для каждого элемента определить: что меняется (полностью или частично).
2. **Определение типа ремонта по каждому элементу.**
   - **Текущий ремонт (замена допускается), если:**
     - Замена отдельного элемента без изменения несущей способности конструктивной схемы.
     - Восстановление работоспособности инженерной системы без её полной замены.
     - Ремонт отделочных покрытий как поддержание эксплуатационных свойств.
   - **Капитальный ремонт (замена относится к капремонту), если:**
     - Замена элемента влечёт изменение несущей способности конструктивной схемы.
     - Полная замена инженерной системы (или её магистральной части).
     - Замена конструкций, относящихся к общему имуществу МКД (по Постановлению Правительства РФ № 491).
3. **Анализ конструктивной схемы (при неоднозначности).**
   - Если замена отдельной стены/колонны/перекрытия — оценить, меняется ли расчётная схема.
   - Если схема не меняется (нагрузки остаются теми же, перераспределения нет) → текущий ремонт.
   - Если схема меняется (демонтаж несущей стены, замена перекрытия на другое по материалу) → капитальный ремонт.
4. **Формулирование выводов.**
   - По каждому виду работ: текущий ремонт / капитальный ремонт.
   - Обоснование со ссылками на нормативные критерии.

## Выходы процесса

- Таблица квалификации видов ремонта (поэлементно)
- Заключение эксперта с обоснованием отнесения работ к текущему или капитальному ремонту

## Оценка успешности

- Каждый вид работ квалифицирован по критериям документа
- Выводы обоснованы ссылками на нормативные акты и проектную документацию

## Если результаты не достигнуты

- **Неоднозначная трактовка (спорная конструкция)** → анализ судебной практики по аналогичным спорам; вывод с альтернативной квалификацией.
- **Отсутствие проектной документации** → вывод о невозможности точной квалификации с указанием неопределённости.
