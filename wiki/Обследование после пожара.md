# Обследование после пожара

Специализированное обследование зданий и сооружений после пожара. Нормативный контур — [[СП 329.1325800.2017]], дополненный рекомендациями и пособиями из `raw/`. Входит в общий цикл по [[ГОСТ 31937-2024]] (п. 4.3: после пожаров и ЧС — обследование обязательно, если есть угроза разрушения).

**Уверенность:** высокая (2 независимых источника: [[СП 329.1325800.2017]] и [[Рекомендации по обследованию после пожара]]).

**Границы применимости:** здания и сооружения после термического воздействия (пожар). **Не** распространяется на технологические/эксплуатационные температурные режимы.

**Ограничения:** для мостовых сооружений после пожара — дополнительно [[Обследование мостовых сооружений]] (СП 79 + ОДМ).

## Входы процесса

- Акт о пожаре (орган ГПН): дата, длительность, температура, площадь горения
- Проектная и исполнительная документация на здание/сооружение
- Результаты предшествующих обследований (если есть)
- Схема расположения очагов пожара и путей распространения огня

## Ресурсы

- **Персонал:** инженер-строитель (конструктор), специалист по пожарно-технической экспертизе
- **Оборудование:** тепловизор, молоток/склерометр (для экспресс-оценки бетона), УЗ-дефектоскоп, щуп, эндоскоп (для скрытых полостей), отборщик кернов, ДТА-анализатор (для определения температуры нагрева)
- **Лаборатория:** испытания образцов на прочность, ДТА (дифференциально-термический анализ)

## Пошаговая инструкция

1. **Определение необходимости обследования ([[ГОСТ 31937-2024]], п. 4.3).**
   - Если есть угроза разрушения или прогрессирующего обрушения → обследование обязательно.
   - Если повреждения локальные и несущие конструкции не затронуты → возможен осмотр без полного обследования.
   - Предварительное обследование комиссией — см. [[Рекомендации по обследованию после пожара#Этапы обследования]] (состав комиссии, срок 1–2 дня, задачи по п. 1.2–1.3).
2. **Визуальный осмотр с фиксацией термических поражений ([[СП 329.1325800.2017]], разд. 5).**
   - Оценка зон термического поражения: цвет, шелушение, оплавление, выкрашивание.
   - Для бетона — цветовая шкала температуры нагрева (приложение Г СП 329).
   - Для стали — деформации, потеря устойчивости, провисание.
   - Для камня — трещины, расслоение, выпадение отдельных камней.
   - Для дерева — обугливание (глубина угольного слоя).
   - Сбор данных о пожаре ([[Рекомендации по обследованию после пожара#Предварительное обследование]], п. 2.3): время обнаружения, продолжительность интенсивного горения, средства тушения, максимальная температура.
3. **Инструментальные измерения прочности после нагрева ([[СП 329.1325800.2017]], разд. 6–9).**
   - Бетон: прочность на сжатие (керны по ГОСТ 28570), УЗ-контроль (ГОСТ 17624), определение температуры нагрева ДТА ([[ГОСТ 31937-2024]], разд. 6).
   - Арматура: прочность на растяжение (образцы по ГОСТ 12004), твёрдость.
   - Сталь: твёрдость, выборочные испытания (ГОСТ 1497, ГОСТ 7564).
   - Камень: прочность по образцам (керны/вырубки).
   - См. также [[Определение температуры нагрева бетона после пожара]].
4. **Определение температуры нагрева конструкций ([[СП 329.1325800.2017]], п. 5.7, приложение Г).**

   **Цвет бетона в зависимости от температуры нагрева:**

   | Температура, °C | Цвет бетона |
   |---|---|
   | до 300 | Розоватый |
   | 300–600 | От серого до красноватого |
   | 600–900 | От тёмно-красного до тёмно-серого |
   | 900–1100 | Коричневый |

   **Коэффициенты снижения прочности бетона при нагреве ([[СП 329.1325800.2017]], п. 11.11):**

   | Коэфф. остаточной прочности | 1,0 | 0,9 | 0,8 | 0,7 | 0,5 | 0,4 | 0,2 |
   |---|---|---|---|---|---|---|---|
   | Температура нагрева, °C | 20 | 120 | 200 | 300 | 400 | 500 | 600 |

   - Для стали — по масштабу окалины (СП 329, п. 3.12), по изменению механических свойств.
   - По косвенным признакам — оплавление металлов (300–500°C — свинец, 400°C — цинк, 650°C — алюминий, 700–850°C — стекло) — см. [[Рекомендации по обследованию после пожара#Определение температуры по косвенным признакам]].
   - Лабораторные методы: ДТА, термолюминесцентный, ртутной порометрии ([[Рекомендации по обследованию после пожара#Оценка параметров пожара]], п. 5.6).
5. **Оценка глубины поражения ([[СП 329.1325800.2017]], разд. 7).**
   - Для бетона — простукивание, УЗ-томография, микроскопия шлифов.
   - Для дерева — замер глубины обугливания щупом (скорость горения 0,04–0,06 см/мин — [[Рекомендации по обследованию после пожара#Определение температуры по косвенным признакам]]).
6. **Классификация состояния ЖБ конструкций после пожара ([[Рекомендации по обследованию после пожара#Классификация состояний ЖБ конструкций после пожара]], разд. 4).**

   | Состояние | Характеристика |
   |---|---|
   | I — нормальное | Усилия в пределах расчётных, дефекты отсутствуют |
   | II — удовлетворительное | Есть дефекты, снижающие долговечность, но не несущую способность |
   | III — непригодное к нормальной эксплуатации | Снижение несущей способности, требуется усиление |
   | IV — аварийное | Нет гарантии сохранности, требуется немедленная разгрузка |

7. **Поверочные расчёты с учётом снижения прочности материалов от нагрева ([[СП 329.1325800.2017]], разд. 10).**
   - Коэффициенты снижения прочности в зависимости от температуры и длительности нагрева (табл. выше, п. 11.11).
   - Если несущая способность >= требуемой → нормативное/работоспособное состояние.
   - Если несущая способность < требуемой → ограниченно-работоспособное или аварийное.
   - Расчёт эквивалентной длительности пожара по стандартной кривой ([[Рекомендации по обследованию после пожара#Оценка параметров пожара]], табл. 5).
8. **Оформление заключения с категорией ТС ([[ГОСТ 31937-2024]], п. 5.1.18).**
   - Нормативное, работоспособное, ограниченно-работоспособное, аварийное.
   - Если аварийное → запрет эксплуатации, мониторинг (см. [[Мониторинг зданий и сооружений]]), проект усиления.
9. **Выбор методов усиления ([[СП 329.1325800.2017]], разд. 11).**
   - Восстановление прочности бетона (торкретирование, инъектирование).
   - Усиление стальных элементов (накладки, обоймы).
   - Замена невосстанавливаемых элементов.
   - См. также [[Каталог конструктивных решений по усилению и восстановлению строительных конструкций зданий и сооружений]].

## Выходы процесса

- Акт визуального осмотра и фотофиксации
- Протоколы измерений (прочность, температура нагрева, глубина поражения)
- Поверочные расчёты с учётом коэффициентов снижения прочности
- Заключение о категории технического состояния
- Проект усиления (при необходимости)

## Оценка успешности

- Определена температура нагрева всех повреждённых конструкций
- Установлена глубина поражения материалов
- Выполнены поверочные расчёты с фактическими прочностями
- Категория ТС присвоена в соответствии с ГОСТ 31937

## Если результаты не достигнуты

- **Не удалось определить температуру нагрева** → принять консервативную оценку (максимальная температура по зоне), пометить как допущение.
- **Керны разрушились при отборе** → использовать НК (УЗ, молоток, эталонные образцы).
- **Поверочный расчёт не сходится** → уточнить нагрузки, провести детальное инструментальное обследование.
- **Аварийное состояние** → см. [[Усиление и восстановление конструкций]] и [[Мониторинг зданий и сооружений]].
