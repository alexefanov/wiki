# Обследование после пожара

Специализированное обследование зданий и сооружений после пожара. Нормативный контур — [[СП 329.1325800.2017]], дополненный рекомендациями и пособиями из `raw/`. Входит в общий цикл по [[ГОСТ 31937-2024]] (п. 4.3: после пожаров и ЧС — обследование обязательно, если есть угроза разрушения).

**Уверенность:** средняя.

**Границы применимости:** здания и сооружения после термического воздействия (пожар). **Не** распространяется на технологические/эксплуатационные температурные режимы.

**Ограничения:** для мостовых сооружений после пожара — дополнительно [[Обследование мостовых сооружений]] (СП 79 + ОДМ).

## Входы процесса

- Акт о пожаре (орган ГПН): дата, длительность, температура, площадь горения
- Проектная и исполнительная документация на здание/сооружение
- Результаты предшествующих обследований (если есть)
- Схема расположения очагов пожара и путей распространения огня

## Ресурсы

- **Персонал:** инженер-строитель (конструктор), специалист по пожарно-технической экспертизе
- **Оборудование:** тепловизор, молоток/склерометр (для экспресс-оценки бетона), УЗ-дефектоскоп, щуп, эндоскоп (для скрытых полостей), отборщик кернов, ДТА-анализатор (для определения температуры нагрева)
- **Лаборатория:** испытания образцов на прочность, ДТА (дифференциально-термический анализ)

## Пошаговая инструкция

1. **Определение необходимости обследования (ГОСТ 31937-2024, п. 4.3).**
   - Если есть угроза разрушения или прогрессирующего обрушения → обследование обязательно.
   - Если повреждения локальные и несущие конструкции не затронуты → возможен осмотр без полного обследования.
2. **Визуальный осмотр с фиксацией термических поражений (СП 329, разд. 5).**
   - Оценка зон термического поражения: цвет, шелушение, оплавление, выкрашивание.
   - Для бетона — цветовая шкала температуры нагрева (розовый → серый → жёлтый → бурый).
   - Для стали — деформации, потеря устойчивости, провисание.
   - Для камня — трещины, расслоение, выпадение отдельных камней.
   - Для дерева — обугливание (глубина угольного слоя).
3. **Инструментальные измерения прочности после нагрева (СП 329, разд. 6–9).**
   - Бетон: прочность на сжатие (керны), УЗ-контроль, определение температуры нагрева ДТА (ГОСТ 31937, разд. 6).
   - Арматура: прочность на растяжение (образцы), твёрдость.
   - Сталь: твёрдость, выборочные испытания.
   - Камень: прочность по образцам (керны/вырубки).
4. **Определение температуры нагрева конструкций (СП 329, п. 5.7).**
   - По цвету бетона, по изменению структуры, по ДТА (керны).
   - Для стали — по масштабу окалины, по изменению механических свойств.
5. **Оценка глубины поражения (СП 329, разд. 7).**
   - Для бетона — простукивание, УЗ-томография, микроскопия шлифов.
   - Для дерева — замер глубины обугливания щупом.
6. **Поверочные расчёты с учётом снижения прочности материалов от нагрева (СП 329, разд. 10).**
   - Коэффициенты снижения прочности в зависимости от температуры и длительности нагрева.
   - Если несущая способность >= требуемой → нормативное/работоспособное состояние.
   - Если несущая способность < требуемой → ограниченно-работоспособное или аварийное.
7. **Оформление заключения с категорией ТС (ГОСТ 31937-2024, п. 5.1.18).**
   - Нормативное, работоспособное, ограниченно-работоспособное, аварийное.
   - Если аварийное → запрет эксплуатации, мониторинг, проект усиления.
8. **Выбор методов усиления (СП 329, разд. 11, или [[Усиление и восстановление конструкций]]).**
   - Восстановление прочности бетона (торкретирование, инъектирование).
   - Усиление стальных элементов (накладки, обоймы).
   - Замена невосстанавливаемых элементов.

## Выходы процесса

- Акт визуального осмотра и фотофиксации
- Протоколы измерений (прочность, температура нагрева, глубина поражения)
- Поверочные расчёты с учётом коэффициентов снижения прочности
- Заключение о категории технического состояния
- Проект усиления (при необходимости)

## Оценка успешности

- Определена температура нагрева всех повреждённых конструкций
- Установлена глубина поражения материалов
- Выполнены поверочные расчёты с фактическими прочностями
- Категория ТС присвоена в соответствии с ГОСТ 31937

## Если результаты не достигнуты

- **Не удалось определить температуру нагрева** → принять консервативную оценку (максимальная температура по зоне), пометить как допущение.
- **Керны разрушились при отборе** → использовать НК (УЗ, молоток, эталонные образцы).
- **Поверочный расчёт не сходится** → уточнить нагрузки, провести детальное инструментальное обследование.
- **Аварийное состояние** → см. [[Усиление и восстановление конструкций]] и [[Мониторинг зданий и сооружений]].
