# Основные положения по проведению экспертиз

Судебная строительно-техническая экспертиза (ССТЭ) — род судебных экспертиз, представляющий собой исследование строительных объектов и территории, функционально связанной с ними, для получения фактических данных, имеющих доказательственное значение в ходе судопроизводства ([[ГОСТ Р 59529-2021]], п. 1). Область применения: уголовные, гражданские и административные дела, а также проверка сообщений о преступлениях.

Страница объединяет правовые основы, бизнес-процесс производства экспертизы и различия между судебной и внесудебной формами. Детальные подпроцессы вынесены на отдельные страницы.

## Судебная vs внесудебная (досудебная) экспертиза

Понятия почти не различаются по существу — применяются одинаковые специальные знания, методики, стандарты. Разница — в процессуальном статусе и правовых последствиях.

| Аспект | Судебная экспертиза | Внесудебная (досудебная) экспертиза |
|--------|---------------------|-------------------------------------|
| Назначение | Определение суда, постановление следователя/дознавателя (ст. 19 [[73-ФЗ]]) → [[Назначение судебной экспертизы ССТЭ]] | Договор с заказчиком (физ/юрлицо) |
| Правовое регулирование | [[73-ФЗ]] + процессуальный кодекс (УПК/ГПК/АПК/КАС/КоАП) | [[73-ФЗ]] (ст. 41 — распространение общих принципов) + договорное право |
| Эксперт | Государственный судебный эксперт (ст. 12 [[73-ФЗ]]) или негосударственный эксперт по поручению суда → [[Права и обязанности эксперта ССТЭ]] | Любое лицо со специальными знаниями |
| Предупреждение об ответственности | За дачу заведомо ложного заключения — ст. 307 УК РФ → [[Ответственность эксперта ССТЭ]] | Как правило, не предупреждается; ответственность — по договору |
| Доказательственное значение | Полноценное доказательство по делу | Иной документ (ст. 89 АПК РФ) → [[Внесудебное заключение в арбитраже]] |
| Статус заключения | «Заключение эксперта» (ст. 9 [[73-ФЗ]]) → [[Заключение эксперта ССТЭ]] | Чаще — «экспертное заключение» или «отчёт специалиста» |
| Осмотр объекта | Суд обеспечивает доступ; стороны извещаются (п. 11 Постановления Пленума ВАС №23) → [[Проведение экспертного осмотра ССТЭ]] | По согласованию с заказчиком |

## Бизнес-процесс: Производство судебной строительно-технической экспертизы

Процесс охватывает полный цикл — от получения постановления (определения) о назначении экспертизы до передачи заключения и материалов дела. Детали каждого этапа — в соответствующих подпроцессах.

### Этапы процесса

| Этап | Краткое описание | Подпроцесс |
|------|-----------------|------------|

| 2. Организация экспертного осмотра | Согласовать дату, известить стороны, подготовить инструменты, провести осмотр | [[Проведение экспертного осмотра ССТЭ]] |
| 3. Натурные исследования | Визуальное и инструментальное обследование, дефектовка, замеры, отбор образцов | [[Комплексное обследование зданий и сооружений]] |

### 4. Камеральная обработка и анализ

**4.1. Обработка результатов измерений:**
- **Геодезические измерения:** расчёт фактических прогибов, осадок, кренов; сравнение с допусками по [[ГОСТ 24846-2019]] (крен — не более 1/500 высоты для большинства зданий, прогиб ж/б балок — не более 1/200 пролёта).
- **Прочность бетона:** статистическая обработка (среднее, СКО, коэффициент вариации) по ГОСТ 18105. Пересчёт в класс бетона по фактической прочности: класс B = средняя прочность × (1 – 1,64 × коэффициент вариации) / 0,778 (для образцов-кубов).
- **Прочность кирпича/камня:** определение марки по ГОСТ 530 (кирпич) или ГОСТ 4001 (камень стеновой) — по среднему и наименьшему значению разрушающей нагрузки.
- **Прочность раствора:** определение марки по ГОСТ 5802 (сжатие образцов-кубов 70,7×70,7×70,7 мм).
- **Обработка результатов НК:** построение градуировочной зависимости (прибор–прочность) при необходимости; пересчёт косвенных показателей в прочность по ГОСТ 22690 или ГОСТ 17624.

**4.2. Сравнение с нормативными требованиями:**

Результаты натурных измерений сравниваются по трём измерениям:

| Параметр                   | С чем сравнивается             | Основание                                                                   |
| -------------------------- | ------------------------------ | --------------------------------------------------------------------------- |
| Геометрические параметры   | Проектная документация, СП     | Отклонения — по СП 70.13330 (несущие конструкции)                           |
| Прочность материалов       | Проектный класс/марка, СП      | Не менее проектного класса (ГОСТ 18105)                                     |
| Дефекты (трещины, прогибы) | Допустимые величины по СП/ГОСТ | Например, ширина раскрытия трещин в ЖБК — не более 0,3–0,5 мм ([[СП 63.13330.2018]]) |
| Объёмы работ               | Договор, акты КС-2, КС-3       | Сопоставление фактических и оплаченных объёмов                              |
| Стоимость работ            | Сметная документация, ТЕР/ФЕР  | Проверка применённых расценок, индексов, объёмов                            |
| Соответствие проекту       | Проектная документация         | Сводка отклонений с оценкой влияния на безопасность                         |

Результаты сводятся в **таблицу сравнения** по каждому вопросу, поставленному судом. Каждая строка таблицы: вопрос → норматив → факт → вывод → обоснование.

**4.3. Причинно-следственный анализ:**

Для вопросов о причинах дефектов/аварий:
1. **Сбор данных:** хронология строительства/эксплуатации, проектные решения, условия эксплуатации, метеоданные, результаты осмотра.
2. **Идентификация дефекта:** вид, параметры, локализация, динамика развития (по актам предыдущих осмотров).
3. **Построение причинно-следственных связей:** от дефекта к возможным причинам через анализ проектных ошибок, нарушений производства работ, нарушений эксплуатации, природных факторов.
4. **Исключение альтернативных причин:** для каждой предполагаемой причины проверяется, могла ли она сама по себе или в совокупности привести к наблюдаемому дефекту.
5. **Формулировка вывода:** о прямой (непосредственной) причине и/или сопутствующих факторах.

**4.4. Сметный расчёт (при вопросах о стоимости):**
- Определение физических объёмов работ по результатам натурных измерений.
- Применение сметно-нормативной базы (ФЕР/ТЕР-2020 с индексацией, либо ресурсным методом).
- Расчёт договорной стоимости выполненных работ (единичные расценки × объём × индекс).
- Расчёт стоимости устранения дефектов (объём дефектных работ × расценка на демонтаж/восстановление).
- Оформление в виде локального сметного расчёта (форма №4 МДС 81-35.2004), прилагаемого к заключению.

### Типовые вопросы эксперту

Группируются по целям исследования (по [[Определение видов, объема, качества и стоимости СМР]]):

1. **Качественные и количественные характеристики работ:** какие виды и объёмы работ выполнены/не выполнены, соответствие проекту, договору, исполнительной документации.
2. **Качество работ:** соответствие требованиям НТД, устранимость дефектов, возможность использования объекта без устранения дефектов.
3. **Стоимость:** стоимость фактически выполненных работ, дополнительных работ, устранения дефектов.
4. **Причинная связь:** связь между действиями и наступившими последствиями (дефектами, авариями).
5. **Прогностические:** тенденция развития дефектов, возможность потери несущей способности.

## Правовая основа

### Федеральные законы

- [[73-ФЗ|Федеральный закон от 31.05.2001 № 73-ФЗ «О государственной судебно-экспертной деятельности в Российской Федерации»]] — определяет правовую основу, принципы организации и основные направления государственной судебно-экспертной деятельности. Является базовым законом для всех экспертиз, включая ССТЭ.
- [[384-ФЗ|Федеральный закон от 30.12.2009 № 384-ФЗ «Технический регламент о безопасности зданий и сооружений»]] — нормирует обязательные требования к зданиям, на соответствие которым проверяются объекты ССТЭ.
- Процессуальные кодексы: [[АПК РФ]], [[ГПК РФ]], [[УПК РФ]], КАС РФ, [[КоАП РФ]] — каждый устанавливает порядок назначения и производства экспертизы в рамках соответствующего вида судопроизводства.

### Разъяснения практики

- [[Постановление Пленума ВАС РФ N 23|Постановление Пленума ВАС РФ от 04.04.2014 № 23 «О некоторых вопросах практики применения арбитражными судами законодательства об экспертизе»]] — разъясняет порядок назначения, проведения, оценки заключения, права сторон, применимость внесудебных заключений в арбитражном процессе.

### Отличие ССТЭ от других судебных экспертиз

ССТЭ отличается от иных родов экспертиз (товароведческой, автотехнической, бухгалтерской) следующим:
- **Объект** — строительные объекты, их элементы, конструкции, строительные работы и материалы.
- **Нормативная база** — строительные нормы ГОСТ, СП, СНиП, технические регламенты ([[384-ФЗ]], [[123-ФЗ]]) в дополнение к процессуальному праву.
- **Методы** — натурное обследование (визуальное + инструментальное), обмеры, поверочные расчёты, сметное нормирование — наряду с общенаучными методами.
- **Специализация эксперта** — высшее строительное образование + ДПО по ССТЭ; от эксперта-строителя требуется знание технологии строительного производства, материаловедения, проектирования, сметного дела.

## Термины и определения

Ключевые термины по судебной экспертизе — в [[terms]] (раздел 24. Судебная экспертиза). Основные:

- **судебная строительно-техническая экспертиза (ССТЭ):** род судебных экспертиз, представляющий собой исследование строительных объектов и территории, функционально связанной с ними, для получения фактических данных, имеющих доказательственное значение ([[ГОСТ Р 59529-2021]], п. 1).
- **судебная экспертиза:** процессуальное действие, включающее проведение исследований и дачу заключения экспертом по вопросам, требующим специальных знаний (ст. 9 [[73-ФЗ]]).
- **заключение эксперта:** письменный документ, отражающий ход и результаты проведённых экспертом исследований (ст. 9 [[73-ФЗ]]).
- **специальные строительно-технические знания:** профессиональные знания в области технологии изготовления и использования строительных материалов, проектирования, возведения, эксплуатации, ремонта ([[ГОСТ Р 59529-2021]], п. 2).

## Методические рекомендации для экспертов

Практические пошаговые руководства по конкретным видам ССТЭ сгруппированы по типовым вопросам (см. «Типовые вопросы эксперту» выше):

### Вопросы качества и объёмов СМР

## Экспертные специальности и аттестация

Судебный эксперт-строитель должен соответствовать требованиям ст. 13 [[73-ФЗ]]:

| Требование | Документальное подтверждение | Периодичность |
|-----------|------------------------------|---------------|
| Высшее строительное образование | Диплом ПГС / промышленное и гражданское строительство / строительство уникальных зданий | Однократно |
| Дополнительное профессиональное образование (ДПО) по экспертной специальности | Удостоверение/диплом о профпереподготовке по ССТЭ | Повышение квалификации — не реже 1 раза в 5 лет |
| Аттестация на право самостоятельного производства экспертизы | Свидетельство об аттестации, выданное экспертно-квалификационной комиссией (ЭКК) судебно-экспертного учреждения | Каждые 5 лет |
| Стаж работы по специальности | Трудовая книжка | Для негосударственных экспертов — не менее 5 лет по специальности |

**Типовые экспертные специальности ССТЭ** (по Приказу Минюста РФ от 27.12.2012 № 237):
- 16.1 — «Исследование строительных объектов и территории, функционально связанной с ними, в том числе с целью проведения их оценки» (основная).
- 16.2 — «Исследование проектной документации, строительных объектов в целях установления их соответствия требованиям специальных правил» (техническое регулирование).
- 16.3 — «Исследование объектов судебной строительно-технической экспертизы с целью определения стоимости» (стоимостная экспертиза).

## Негосударственная судебная экспертиза

Согласно ст. 41 [[73-ФЗ]], действие закона распространяется и на негосударственных судебных экспертов. Подробнее о внесудебном заключении в арбитраже — [[Внесудебное заключение в арбитраже]].

| Аспект | Государственный эксперт | Негосударственный эксперт |
|--------|------------------------|--------------------------|
| Трудоустройство | В штате ГСЭУ (Минюст, МВД, МЧС, ФСБ и др.) | ИП или сотрудник коммерческой организации |
| Основание назначения | Определение суда / постановление следователя | Определение суда / договор с заказчиком (внесудебная) |
| Аттестация | Внутренняя ЭКК ГСЭУ | ЭКК при Минюсте РФ (Приказ №237) или СРО |
| Материально-техническая база | Государственная (лаборатории, оборудование, поверка) | Собственная или на условиях аренды |
| Оплата | Из бюджета (ст. 132 УПК, ст. 96 ГПК) | По договору с заказчиком или за счёт стороны (ст. 109 АПК) |
| Ответственность | Дисциплинарная + уголовная (ст. 307 УК) | Уголовная (ст. 307 УК) + гражданско-правовая (договор) |
| Сроки | Регламентированы ведомственными нормами | По договору или соглашению с судом |

### Условия назначения негосударственного эксперта ([[Постановление Пленума ВАС РФ N 23|Постановление ВАС №23]], п. 2)

Суд вправе назначить негосударственного эксперта, если:
- У государственных экспертов отсутствует необходимая специальность.
- Государственные эксперты не могут дать заключение в установленный срок.
- Сторона заявила ходатайство о конкретном негосударственном эксперте (с обоснованием его компетентности).
- Предмет исследования требует уникальных знаний/оборудования, отсутствующих в ГСЭУ.

Эксперт-негосударственник должен представить суду: документы о высшем строительном образовании, ДПО по экспертной специальности, свидетельство об аттестации, сведения о стаже, копии свидетельств о поверке приборов (по запросу суда).

### Риски негосударственного эксперта

- **Оспаривание компетентности** — сторона может заявить отвод (→ [[Отвод эксперта ССТЭ]]). Рекомендация: заранее представить суду все документы об образовании и аттестации.
- **Давление стороны** — заказчик (плательщик) может пытаться влиять на выводы. Эксперт обязан предупредить заказчика, что заключение даётся на научной основе, независимо от пожеланий. При давлении — самоотвод.
- **Уголовная ответственность (ст. 307 УК)** — негосударственный эксперт предупреждается об ответственности наравне с государственным (→ [[Ответственность эксперта ССТЭ]]).
- **Гражданско-правовая ответственность** — заказчик может взыскать убытки, если из-за некачественного заключения проигран суд. Рекомендация: страхование профессиональной ответственности эксперта.

## Стоимость производства экспертизы

Стоимость определяется:
- **Для государственных экспертов:** по калькуляции, утверждённой ведомственными нормами (Постановление Правительства РФ от 01.12.2012 № 1240).
- **Для негосударственных:** по договорённости с заказчиком или на основании запроса суда. Сторона, заявившая ходатайство, вносит деньги на депозит суда (ст. 108 АПК РФ, ст. 96 ГПК РФ).

Факторы, влияющие на стоимость ССТЭ:

| Фактор | Влияние на стоимость |
|--------|---------------------|
| Объём объекта | Количество конструкций, этажей, помещений |
| Сложность | Наличие уникальных конструкций, историческая ценность, агрессивная среда |
| Доступность | Высота, подвалы, удалённость, необходимость лесов/подмостей |
| Количество методов | Визуальный, инструментальный, поверочные расчёты, лабораторные испытания |
| Срочность | Коэффициент 1,5–2,0 при сжатых сроках |
| Командировочные расходы | Транспорт, проживание, суточные |
