# Заключение эксперта ССТЭ

Типовой подпроцесс: форма, структура, требования к заключению судебного эксперта-строителя по кодексам и [[73-ФЗ]].

## Входы процесса
- Результаты осмотра объекта ([[Проведение экспертного осмотра ССТЭ]]).
- Результаты лабораторных испытаний (если проводились).
- Поверочные расчёты (прочность, несущая способность, сметные).

## Ресурсы
- Эксперт (один или несколько при комиссионной/комплексной экспертизе).
- Программное обеспечение для расчётов (SCAD, ЛИРА, ГРАНД-Смета).
- Нормативные документы: ГОСТ, СП, методики.

## Структура заключения (ст. 25 [[73-ФЗ]])

1. **Вводная часть:**
   - Дата, время, место производства экспертизы.
   - Основания (постановление / определение, номер, дата, кем вынесено).
   - Сведения об эксперте: ФИО, образование, специальность, стаж, номер свидетельства об аттестации.
   - Предупреждение об ответственности по ст. 307 УК РФ (с подпиской).
   - Вопросы, поставленные перед экспертом (дословно).
   - Объекты и материалы, представленные для исследования.

2. **Исследовательская часть:**
   - Описание объекта (адрес, конструктивная схема, этажность, год постройки).
   - Методы исследования: визуальный осмотр, инструментальные замеры, лабораторные испытания, поверочные расчёты (со ссылками на ГОСТ/СП/методики).
   - Результаты осмотра: описание дефектов с привязкой к осям, фотофиксация (по каждой конструкции).
   - Результаты замеров: прочность бетона, параметры трещин, отклонения от вертикали/горизонтали (таблицы, графики).
   - Результаты поверочных расчётов: схемы нагрузок, запасы прочности.
   - Оценка категории ТС: работоспособное, ограниченно-работоспособное, аварийное (по [[ГОСТ 31937-2024]]).

3. **Выводы:**
   - Краткие, чёткие ответы на каждый поставленный вопрос.
   - Каждый вывод должен соответствовать исследовательской части (проверка на противоречие).

4. **Приложения:**
   - Фототаблица (каждое фото с подписью: ось, этаж, элемент, направление съёмки).
   - Замерные листы (протоколы испытаний).
   - Копии свидетельств о поверке приборов.
   - Схемы, чертежи, графики.

## Требования по кодексам

| НПА | Основные требования |
|-----|---------------------|
| ст. 25 [[73-ФЗ]] | Обязательные разделы, подпись эксперта, печать ГСЭУ |
| ст. 204 [[УПК РФ]] | Содержание, подпись, приложения |
| ст. 86 [[ГПК РФ]] | Письменная форма, подпись, приложения |
| ст. 86 [[АПК РФ]] | Письменная форма, подпись, приложения |

## Типовые ошибки

| Ошибка | Последствие | Как избежать |
|--------|-------------|-------------|
| Выводы выходят за пределы специальных знаний | Заключение не принимается (ст. 8 [[73-ФЗ]]) | Формулировать в терминах тех. соответствия |
| Нет ссылок на методики | Непроверяемость | Каждое измерение — со ссылкой на ГОСТ/СП |
| Измерения неповеренными приборами | Недостоверность | Прикладывать копии свидетельств |
| Неполный ответ на вопрос | Назначение дополнительной экспертизы | Разбить вопрос на подвопросы; проверить покрытие |
| Фототаблица без привязки | Невозможность идентификации | Каждое фото — с подписью |
| Вывод противоречит исследовательской части | Возврат заключения | Проверка: каждый вывод следует из данных |
