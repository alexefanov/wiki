# LLM-Wiki Index

Карта обработанной базы знаний по строительному обследованию, экспертизе и контролю качества.

## Для агента

Правила поведения для LLM, поддерживающего этот vault, — в `AGENTS.md` (корень репозитория). Кратко:

1. **Порядок поиска ответа:** `wiki/` → `raw/` → web. Начинай с этой страницы. Если обращаешься в web — предупреди.
2. **Термины:** используй определения из [[terms]]; не заменяй близкими по смыслу.
3. **Ссылки:** цитируй источники для нетривиальных утверждений.
4. **Crystallization:** если родился новый устойчивый вывод — предложи сохранить как wiki-страницу.

Связанные страницы: [[Source Notes]], [[biblio]], [[terms]], [[log]].
