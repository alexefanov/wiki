# Определение видов, объема, качества и стоимости СМР

Методика (подпроцесс судебной строительно-технической экспертизы) по установлению фактических видов, объёмов, качества и стоимости строительно-монтажных и специальных работ. Используется при договорных спорах, оспаривании актов КС-2, определении объёма невыполненных работ.

**Авторы:** А.Ю. Бутырин, А.Р. Чудиёвич, О.В. Луковкина.

**Уверенность:** средняя.

**Границы применимости:** здания, строения, сооружения (включая благоустройство, инженерные сети), оконченные строительством, в стадии возведения либо ремонта; объекты незавершённого строительства.

## Входы процесса

- Договор строительного подряда (контракт)
- Проектная и рабочая документация (разделы КР, АР, ИОС)
- Исполнительная документация: акты КС-2, КС-3, КС-6, КС-11, журналы работ
- Акты скрытых работ, протоколы испытаний, сертификаты на материалы
- Дефектная ведомость (при спорах о качестве)
- Сметная документация (ТЕР, ФЕР, ресурсный метод)

## Ресурсы

- **Персонал:** эксперт-строитель, сметчик
- **Оборудование:** рулетка, лазерный дальномер, толщиномер, уровнемер, фотоаппарат
- **ПО:** сметные программы (Гранд-Смета, РИК), AutoCAD (сверка с проектом), Excel

## Пошаговая инструкция

1. **Изучение материалов дела.**
   - Договор, проект, исполнительная документация.
   - Акты КС-2 (проверить подписи, объёмы, расценки).
2. **Натурное обследование объекта.**
   - Визуальный осмотр, обмеры, фотофиксация.
   - Вскрытие конструкций (при необходимости) — для проверки скрытых работ.
   - Составление ведомости фактических объёмов.
3. **Сравнение фактических объёмов с проектными и договорными.**
   - Если факт < проект → невыполненные работы.
   - Если факт > проект → дополнительные работы.
   - Если факт = проект → работы выполнены в полном объёме.
4. **Оценка качества работ.**
   - Сверка с требованиями НТД (ГОСТ, СП, ТУ, СНиП).
   - Если дефекты есть → классификация: устранимые / неустранимые / критические.
   - Если дефекты критические → работы не соответствуют договору.
5. **Определение стоимости.**
   - Стоимость фактически выполненных качественных работ.
   - Стоимость дополнительных работ (если были согласованы).
   - Стоимость устранения дефектов (сметный расчёт).
   - Стоимость невыполненных работ (исключение из актов КС-2).
6. **Формулирование выводов.**
   - Какие виды и объёмы работ выполнены/не выполнены.
   - Качество выполненных работ (соответствие/несоответствие НТД).
   - Стоимость фактически выполненных работ, устранения дефектов, невыполненных работ.

## Выходы процесса

- Ведомость фактических объёмов работ
- Дефектная ведомость (при несоответствии качества)
- Сметный расчёт стоимости выполненных работ и устранения дефектов
- Заключение эксперта с выводами по каждому из поставленных вопросов

## Оценка успешности

- Все поставленные вопросы разрешены: виды, объёмы, качество, стоимость
- Фактические объёмы подтверждены натурными обмерами
- Качество проверено по критериям действующей НТД
- Стоимость рассчитана по утверждённой сметно-нормативной базе

## Если результаты не достигнуты

- **Доступ на объект отсутствует** → оценка по документации с пометкой «при условии достоверности исполнительной документации».
- **Проектная документация утрачена** → экспертное восстановление проекта на основе натурных обмеров и справочных данных.
- **Дефекты не классифицируются однозначно** → назначение дополнительной экспертизы (материаловедческой, проектной).
